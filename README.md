# Solid Tetris

This is a simple Tetris game written in TS/[Solidjs](https://solidjs.com). 

Just to practise and fun.

## Usage

Pretty standard workflow:

``` bash
* git clone repo
* cd into/folder
* npm install
* npm run dev
```

## Playing

Pretty standard tetris:

* Arrows left/down/right to move,
* Arrow up to rotate
* P/p to pause
